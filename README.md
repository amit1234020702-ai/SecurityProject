# Security Project

A Spring Boot based Security Project that provides secure user authentication and authorization using Spring Security and JWT (JSON Web Token).

## Features

* User Registration and Login
* JWT Token Generation and Validation
* Secure REST APIs
* Password Encryption
* MySQL Database Integration
* Authentication and Authorization using Spring Security

## Technologies Used

* Java 17
* Spring Boot
* Spring Security
* JWT
* Spring Data JDBC
* MySQL
* Maven

## Project Structure

* Controller Layer
* Service Layer
* Repository Layer
* Security Configuration
* Database Layer

## Purpose

The main purpose of this project is to learn and implement secure authentication and authorization mechanisms in Spring Boot applications using JWT and Spring Security.
